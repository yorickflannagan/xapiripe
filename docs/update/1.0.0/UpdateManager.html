UpdateManager(nodeProcess, distribution, callback)

Constructor

new UpdateManager(nodeProcess, distribution, callback)

Cria uma nova instância do gerenciador de atualização
Parameters:
Name Type Description
nodeProcess Process instância do objeto global process
distribution Distribution identificação da distribuição do produto
callback function função a ser chamada quando houver necessidade de comunicação com o processo principal da aplicação, durante o processo contínuo de verificação de atualização. Deve ter a assinatura (type, message), a saber: - type: tipo de mensagem (ver propriedades estáticas da classe); - message: mensagem a ser exibida ao usuário. Opcional; - retorna um indicador de que a ação requerida pode prosseguir.
