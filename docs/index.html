Xapiripë – Componentes web para assinatura digital

Xapiripë é o projeto para um conjunto de componentes destinados fornecer para aplicações web serviços de emissão de certificados e assinaturas digitais cujas chaves estejam residentes em dispositivos criptográficos de hardware (smart cards) e software (repositório do Windows). Ele nasce como uma revisão da solução Kriptonita hoje disponível como um conjunto de extensões de navegador na Chrome Web Store.

Ainda que a maioria dos navegadores já tenha implementado a especificação do W3C Web Cryptography API, a necessidade de um tal suplemento permanece: por razões de segurança, as implementações da API não interagem com smart cards e outros dispositivos criptográficos locais.
