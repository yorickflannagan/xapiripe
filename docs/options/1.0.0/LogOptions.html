LogOptions()

Opções de log

Constructor

new LogOptions()

Properties:
Name Type Description
path: String caminho completo para os arquivos de log. Default: __dirname.
fname: String padrão para nome dos arquivos de log. Deve conter a string "-n". Default: xapiripe-n.log,
maxSize: Number tamanho máximo (em KB) que o arquivo de log pode alcançar antes de rotacionar. Default: 2048.
rotate: Number quantidade máxima de arquivos de log a serem criados antes de sobrescrever o mais antigo. Default: 5.
level: Number nível de log a ser adotado pela aplicação. Default: 1 (info). Ver a documentação do componente Wanhamou.
