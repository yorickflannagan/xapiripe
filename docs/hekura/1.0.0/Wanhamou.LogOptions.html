Opções de inicialização do log
Properties:
Name Type Description
path String Diretório de localização do arquivo de log. Valor default __dirname
fname String Padrão de nome do log, na forma [nome]-n.[ext], onde nome é o nome e ext a extensão que se deseja para o arquivo. Valor default xapiripe-n.log
maxSize Number Tamanho máximo (em KB) do arquivo de log antes de ser obrigado a rotacionar. Valor default: 2048
rotate Number Quantidade máxima de arquivos de log antes que seja necessário sobrescrever o mais antigo. Valor default: 5
level Number Nível corrente do log. Serão logados somente as mensagens com valor igual ou maior. Valor default INFO
