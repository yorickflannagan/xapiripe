Class: CAdES

Aroari.CAdES()

new CAdES()

Opções CAdES da assinatura. Atributos assinados obrigatórios: Content Type, Message Digest e ESS signing-certificate-v2
Properties:
Name Type Description
policy String Padrão de assinatura escolhido conforme a RFC 5126. Opcional. Valor default: CAdES-BES
addSigningTime Boolean Incluir atributo assinado Signing Time. Opcional. Valor default: true
commitmentType String Se contiver um valor descritivo, inclui o OID do atributo assinado Commitment Type Indication conforme Aroari.CommitmentType. Opcional.
