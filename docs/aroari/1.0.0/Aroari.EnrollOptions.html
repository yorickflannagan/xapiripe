Class: EnrollOptions

Aroari.EnrollOptions()

new EnrollOptions()

Opções para a geração de request de certificados.
Properties:
Name Type Description
device String Cryptographic Service Provider ou Key Storage Provider que a ser utilizado para gerar as chaves RSA. Deve corresponder exatamente a um dos dispositivos retornados por Aroari.enumerateDevices
keySize Number Tamanho (em bits) das chaves RSA a serem geradas. Opcional. Default: 2048
signAlg Number Algoritmo a ser utilizado na assinatura da requisição de certificado. Opcional. Default: SignMechanism.CKM_SHA256_RSA_PKCS
rdn Object Nome distinto do titular do certificado, conforme Aroari.X500Name. Obrigatório
