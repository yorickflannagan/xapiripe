Class: API

API()

Fornece acesso às APIs Xapiripe ou Kryptonite. As três últimas propriedades só são retornadas caso seja desejado o modo de compatibilidade com a Kryptonite.

Constructor

new API()

Não exportado
Properties:
Name Type Description
signet String Identificador do cliente utilizado, isto é: Xapiripe ou Kryptonite
enroll Object Instância do objeto Enroll. No caso da API Kryptonite esta propriedade pode estar indefinida, caso o usuário não tenha instalado a extensão emissão de certificados digitais. No caso da API Xapiripe, a propriedade é sempre definida.
sign Object Instância do objeto Sign. No caso da API Kryptonite esta propriedade pode estar indefinida, caso o usuário não tenha instalado a extensão de assinatuas digitais. No caso da API Xapiripe, a propriedade é sempre definida.
verify Object Instância do objeto Verify. No caso da API Kryptonite esta propriedade pode estar indefinida, caso o usuário não tenha instalado a extensão verificação de assinaturas digitais. No caso da API Xapiripe, a propriedade é sempre definida.
base64 Object Instância do objeto Base64. Esta propriedade só estará definida se a função queryInterface for executada no mode de compatabilidade.
deflater Object Instância do objeto Deflate. Esta propriedade só estará definida se a função queryInterface for executada no mode de compatabilidade.
inflater Object Instância do objeto Inflate. Esta propriedade só estará definida se a função queryInterface for executada no mode de compatabilidade.

Methods

matchDN(dna, dnb)

Verifica se dois distinguished names são o mesmo DN. O método é capaz de comparar duas representações distintas de DNs, desde que atendam à RFC 1779.
Parameters:
Name Type Description
dna String DN que se quer validar
dnb string DN contra o qual se quer comparar
