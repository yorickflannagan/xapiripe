Class: SignOptions

Aroari.SignOptions()

new SignOptions()

Opções para a operação de assinatura de documentos e transações
Properties:
Name Type Description
handle Number Handle para o certificado, retornado por Xapiripe.Certificate
toBeSigned String | ArrayBuffer Documento ou transação a ser assinada
attach Boolean Indica se o documento toBeSigned deve ser anexado ao envelope CMS Signed Data. Opcional. Valor default: true
algorithm Number Constante indicativa do algoritmo de assinatura a ser utilizado. Valor default: SignMechanism.CKM_SHA256_RSA_PKCS
policy String Padrão de assinatura escolhido conforme a RFC 5126. Opcional. Valor default: CAdES-BES
