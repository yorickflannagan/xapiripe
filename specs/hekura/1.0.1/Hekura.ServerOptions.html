Opções de inicialização do servidor Hekura
Properties:
Name Type Description
port Number Porta de atendimento. Se indefinido, assume o valor 9171
maxAge Number O valor em segundos que os preflight requests do protocolo CORS podem ser cacheados. Se indefinido, assume o valor 1800
cors Object Implementação de bloqueio de origens não confiáveis. Se definido deve ser uma herança de Hekura.CORSBlockade. Caso o parâmetro não seja informado ou o objeto não seja uma instância de CORSBlockade, esta impementação é utilizada silenciosamente
callback approvalCallback Função a ser chamada visando obter a aprovação do usuário para a operação criptográfica. Visando simplificar os testes de regressão, a propriedade assume como default uma função que sempre retorna true. No entanto, em vista do modelo de ameaças da solução, é esperado que um valor seja passado.
